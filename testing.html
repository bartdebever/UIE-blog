Result

We made the testers purchase a game, install said game and send a message to a friend.

  • One tester didn’t really understand how purchasing a game worked, he was confused about the payment being an external piece of software so we wouldn’t have a screen for it in our prototype. This was mostly my fault as I didn’t really explain that the purchasing wasn’t done internally.
  • When trying to install said game he wanted to view order history to install the game, we didn’t have this option and might have to add this under profile.
  • Sending messages went just right, tester pressed the Friends icon instead of going to Profile > Friends which means the icon was a good addition.
