Conclusion

From these test we came to the following feedback:

  • Tester had trouble enabling settings because the accent color was the same as the background color
  • Tester wasn't sure what ‘remote install’ did or what it meant
  • Tester couldn't find an option to change language
  • Testers were using the hamburger menu a lot, which made it a powerful tool
  • Tester was confused about the difference between library and store

Thanks to this feedback we came to some conclusions

  • Most of the features were easy to find/use, but the ability to change the language of the app itself, not the whole system should be included, so users can change the language without changing the systemlanguage.
  • The language menu should be findable in the settings and the friends should also be clickable in the profile. The swipe to the right menu should be seen as a very powerful tool.
  • User was able to find every task fairly easily. Wanted to play the game after installing it, but this isn't possible because it installed on their PC or laptop.
  • User had some trouble disabling the notifications because the color of the toggle is the same as the background.
